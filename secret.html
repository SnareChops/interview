Proctor Instructions

The point of this evaluation is to see how the candidate handles debugging a malformed data source. The candidate is given the following instructions:

Available at /data is a list of fruits. Display this list of fruits in an unordered list, sorted alphabetically.

You are free to use any comfortable language, framework, etc. You may use google, LLMs, StackOverflow, or any other resource you would like as if this were a real task. You are free to ask any questions during the task, and it is recommended to talk through your thought processes.

Styling is completely optional and not being evaluated in this task

If you encounter any issues during the task, take us through your debugging processes and use any and all tools and resources.

The data at this endpoint is malformed and is invalid json (missing a comma). This is intentional and is meant to challenge the candidate to discover and correctly identify the malformed json. Once the candidate has shown competence in debugging this issue and identifying the root cause, supply the candidate with the corrected endpoint /data/v2. This corrected endpoint should allow them to complete the originally assigned task.
